## 🔧 Como Funciona o Cálculo

### **Implementação Técnica**

O sistema utiliza a classe `ServicePricing.kt` que contém:

1. **Busca Específica**: Primeiro tenta encontrar o preço exato do serviço
2. **Busca Case-Insensitive**: Se não encontrar, tenta busca ignorando maiúsculas/minúsculas
3. **Preço Padrão**: Se não encontrar nada, usa o preço padrão da categoria

### **Exemplo de Uso**

```kotlin
// Cliente seleciona:
Categoria: "Elétrica"
Serviço: "Instalação de lâmpadas"

// Sistema calcula:
Valor: R$ 145,00

// No pagamento:
Cliente paga: R$ 145,00 via Pagar.me
```
