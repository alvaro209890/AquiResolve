# Site de Política de Privacidade — AquiResolve

Esta pasta contém o site estático da Política de Privacidade exigida pelo Play Console.

## Arquivos

- `index.html` — página principal da política.
- `styles.css` — estilos da página.

## Como publicar

Pode ser hospedado em qualquer serviço estático, por exemplo:

- Firebase Hosting
- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages

Depois de publicar, use a URL pública no Play Console em:

`Conteúdo do app → Política de Privacidade`

## Observação

A política foi escrita para refletir o funcionamento atual do AquiResolve: Firebase, Pagar.me, OSMDroid/OpenStreetMap, OSRM, localização do prestador e notificações.
